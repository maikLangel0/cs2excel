<h3>Building from source</h3>
<p>Make sure you have git, rust and cargo installed.</p>
<p>Clone the repo:</p>

```
git clone https://github.com/maikLangel0/cs2excel.git
```

<p>Build:</p>

```
cargo build --release
```
